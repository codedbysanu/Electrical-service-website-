# Shanza Electrical Solutions — Business Website

A multi-page business website built with pure HTML5 and CSS3.

## Live Site
shanzaelectricalsolutions.netlify.app/project.html

## Pages
- project.html — Landing page with hero section and navigation
- service.html — Services overview table linking to 6 sub-pages
- service1–6.html — Individual detailed service pages
- contact.html — Contact info, business cards display, feedback form

## Tech Stack
- HTML5 (semantic structure, multi-page architecture)
- CSS3 (all styling written from scratch, no frameworks)
- Google Fonts — Orbitron & Montserrat
- Hosted on Netlify

## CSS Techniques Used
- Flexbox for layout and alignment
- CSS transitions and hover animations
- Glow effects using box-shadow with rgba
- Text-shadow for neon styling
- Responsive design with max-width and flex-wrap
- CSS pseudo-classes (:hover, :focus)
- rgba transparency for layered backgrounds
- CSS reset using universal selector

## Features
- Fully responsive multi-page navigation
- Animated hover effects on cards, buttons, and table rows
- Contact form with HTML5 input validation (required fields)
- Business card image display with hover lift effect
- Consistent dark cyber-themed design across all pages
- Inter-page linking architecture (9 total HTML files)

## Design Style
Dark background (#02050a) with cyan accent (#00d4ff),
creating a tech/industrial aesthetic matching the 
electrical engineering brand.
